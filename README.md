Telegram Бот для клиники "Альтернатива"

Бот для клиники "Альтернатива", предоставляющий информацию о специалистах, услугах и направлениях медицинского центра.

1. Установка 

    1.1 Клонируйте репозиторий: git clone <url_репозитория>
    1.2 Откройте данный проект в PyCharm.
    1.2 Установите зависимости:
    pip install > requirements.txt

2. Запуск и проверка

    2.1 Запустите бота с помощью команды python (или python3) bot.py
    2.2 Перейдите по ссылке в телеграм на данного бота и проверьте его работу.
